Climate Analytics Project

Climate Data Analysis and Rainfall Prediction with Dimensionality Reduction and Machine Learning

A report-driven study of Australian weather data using PCA, SVD, K-Means clustering, and predictive models to uncover climate structure and improve rainfall prediction.

Read Final Report Open Merged Code View Presentation

What This Project Delivers

The project reduces a high-dimensional climate dataset into interpretable structure, identifies distinct weather regimes, and compares predictive methods for rainfall-related outcomes.

>95% variance retained by PCA according to the final report
3 weather-pattern clusters identified through K-Means
PCA > SVD better final dimensionality-reduction performance in this project
ML + EDA combined statistical exploration, clustering, and predictive modeling

Approach

  • Cleaned and prepared climate observations from an Australian weather dataset.
  • Engineered interaction and lag-based features to capture rainfall and temperature dynamics.
  • Applied PCA and SVD to reduce dimensionality and improve interpretability.
  • Used K-Means clustering to expose natural groupings in weather behavior.
  • Compared classification and regression models to evaluate predictive performance.

Highlights

  • Strong rainfall seasonality visible across monthly and geographic breakdowns.
  • Clear relationships between rainfall, sunshine, temperature, humidity, and wind.
  • Distinct cluster structure in reduced feature space.
  • Ensemble models delivered strong rainfall-prediction performance in the final report.
